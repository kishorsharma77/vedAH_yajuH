(अथ षष्ठाष्टके तृतीयप्रपाठके चतुर्थोऽनुवाकः )।  
पृ॒थि॒व्यै त्वा॒ऽन्तरि॑क्षाय त्वा दि॒वे त्वेत्या॑है॒भ्य  
ए॒वैनं॑ लो॒केभ्यः प्रोक्ष॑ति॒ परा॑ञ्चं॒ प्रोक्ष॑ति॒ परा॑ङिव॒  
हि सु॑व॒र्गो लो॒कः क्रू॒रमि॑व॒ वा ए॒तत्क॑रोति॒ यत्खन॑-  
त्य॒पोऽव॑ नयति॒ शान्त्यै॒ यव॑मती॒रव॑ नय॒त्यूर्ग्वै यवो॒  
यज॑मानेन॒ यूपः॒ संमि॑तो॒ यावा॑ने॒व यज॑मान॒स्ताव॑ती-  
मे॒वास्मि॒न्नूर्ज॑ दधाति ( १ ) पि॒तृ॒णाꣳ सद॑नम॒सीति॑  
ब॒र्हिरव॑ स्तृणाति पितृदेव॒त्या॑ १॒ꣳ ह्ये॑तद्यन्निखा॑तं॒  
यद्ब॒र्हिरन॑वस्तीर्य मिनु॒यात्पि॑देव॒त्यो॑ निखा॑तः  
स्याद्ब॒र्हिर॑व॒स्तीर्य॑ मिनोत्य॒स्यामे॒वैनं॑ मिनोति यूप-  
शक॒लमवा॑स्यति॒ सते॑जसमे॒वैनं॑ मिनोति दे॒वस्त्वा॑  
सवि॒ता मध्वा॑ऽन॒क्त्वित्या॑ह॒ तेज॑सै॒वैन॑मनक्ति सुपि-  
प्प॒लाभ्य॒स्त्वौष॑धीभ्य॒ इति॑ चपालं॒ प्रति॑ ( २ )  
मु॒ञ्च॒ति॒ तस्मा॑च्छीर्ष॒त ओष॑धयः॒ फलं॑ गृह्णन्त्य॒नक्ति॒  
तेजो॒ वा आज्यं॒ यज॑मानेनाग्नि॒ष्ठाऽश्रिः॒ संमि॑ता॒  
यद॑ग्नि॒ष्ठामश्रि॑म॒नक्ति॒ यज॑मानमे॒व तेज॑साऽऽनक्त्या॒-  
न्तम॑नक्त्या॒न्तमे॒व यज॑मानं॒ तेज॑साऽनक्ति स॒र्वतः॒  
परि॑ मृश॒त्यप॑रिवर्गमे॒वास्मि॒न्तेजो दधा॒त्युद्दिव॑ꣳ  
स्तभा॒नाऽन्तरि॑क्षं पृ॒णेत्या॑है॒षां लो॒कानां॒ विधृ॑त्यै

[प्रपा॰ ३ अनु॰ ४] कृष्णयजुर्वेदीयतैत्तिरीयसंहिता।  
( यूपस्थापनाभिधानम् )  
वैष्ण॒व्यर्चा ( ३ ) क॒ल्प॒य॒ति॒ वै॒ष्ण॒वो वै दे॒वत॑या॒  
यूपः॒ स्वय॒वैन॑ दे॒वत॑या कल्पयति॒ द्वाभ्या॑ कल्पयति  
द्वि॒पाद्यज॑मानः॒ प्रति॑ष्ठित्यै॒ यं का॒मये॑त॒ तेज॑सैनं  
दे॒वताभिरिन्द्रि॒येण॒ व्य॑र्धयेय॒मित्य॑ग्नि॒ष्ठां तस्याश्चि॑-  
माहव॒नीया॑दि॒त्थं वेत्थं वाऽति॑ नावये॒त्तेज॑सै॒वैनं॑  
दे॒वता॑भिरिन्द्रि॒येण॒ व्य॑र्धयति॒ यं का॒मये॑त॒ तेज॑सैनं  
दे॒वता॑भिरिन्द्रि॒येण॒ सम॑र्धयेय॒मिति॑ ( ४ ) अ॒ग्रि॒ष्ठां  
तस्याश्रि॑माहव॒नीये॑न॒ सं मि॑नुया॒त्तेज॑सै॒वैनं॑ दे॒व-  
ता॑भिरिन्द्रि॒येण॒ सम॑र्धयति ब्रह्म॒वनिं॑ त्वा क्षत्र॒-  
वनि॒मित्या॑ह यथाय॒जुरे॒वैतत्परि॑ व्यय॒त्यूर्ग्वै र॑श॒ना  
यज॑मानेन॒ यूपः॒ संमि॑तो॒ यज॑मानमे॒वोर्जा सम॑-  
र्धयति नाभिद॒घ्ने परि॑ व्ययति नाभिद॒घ्न ए॒वास्मा॒  
उ॑र्जं॑ दधाति॒ तस्मा॑न्नाभिदघ्न उ॒र्जा भु॑ञ्जते॒ यं  
का॒मये॑तो॒र्जैन॑म् ( ५ ) व्य॑र्धयेय॒मित्यू॒र्ध्वां वा॒ तस्या-  
वा॑चीं॒ वाऽवो॑हेदू॒र्जैवैनं॒ व्य॑र्धयति॒ यदि॑ का॒मये॑त॒  
वर्षु॑कः प॒र्जन्यः॑ स्या॒दित्यवा॑ची॒मवो॑हे॒द्वृष्टि॑मे॒व नि  
य॑च्छति॒ यदि॑ का॒मये॒ताव॑र्षुकः स्या॒दित्यू॒र्ध्वामुदू॑हे॒-  
द्दृष्टि॑मे॒वोद्य॑च्छति पितृ॒णां निखा॑तं मनुष्या॑णामू॒र्ध्वं  
निखा॑ता॒दा र॑श॒नाया॒ ओष॑धीनाꣳ रश॒ना विश्वे॑-  
षाम् ( ६ ) दे॒वाना॑मूर्ध्वꣳ र॑श॒नाया॒ आ च॒षाला॒दि-  
न्द्र॑स्य च॒षा॑लꣳ सा॒ध्याना॒मति॑रिक्त॒ꣳ स वा ए॒ष  
स॑र्वदेव॒त्यो॑ यद्युपो॒ यद्यूपं॑ मि॒नोति॒ सर्वा॑ ए॒व दे॒वताः॑  
प्रीणाति य॒ज्ञेन॒ वै दे॒वाः सु॑वर्गं लो॒कमा॑यन्ते॑ऽमन्यन्त

श्रीमत्सायणाचार्यविरचितभाष्यसमेतमा— [ ६ षष्ठकाण्डे---  
( अग्निमन्थनाभिधानम् )  
मनु॒ष्या॑ नो॒ऽन्वाभ॑विष्य॒न्तीति॒ ते यूपे॑न योपयि॒त्वा  
सु॑व॒र्गं लो॒कमा॑य॒न्तमृष॑यो॒ यूपे॑नै॒वानु॒ प्राजा॑न॒न्तद्यू-  
प॑स्य यूप॒त्वम् ( ७ ) यद्यू॑पं मि॒नोति॑ सुव॒र्गस्य॑  
लो॒कस्य॒ प्रज्ञा॑त्यै पु॒रस्ता॑न्मिनोति पु॒रस्ता॒द्धि य॒ज्ञस्य॑  
प्रज्ञा॒यते प्र॑ज्ञात॒ꣳ हि तद्यदति॑पन्न आ॒हुरि॒दं का॒र्य॑-  
मासी॒दिति॑ सा॒ध्या वै देवा य॒ज्ञमत्य॑मन्यन्त॒ तान्य॒ज्ञो  
नास्पृ॑श॒त्तान्यद्य॒ज्ञस्याति॑रिक्त॒मासी॒त्तद॑स्पृश॒दति॑रिक्तं॒  
वा ए॒तद्य॒ज्ञस्य॒ यद॒ग्नाव॒ग्निं म॑थि॒त्वा प्र॒हर॒त्यति॑रिक्त-  
मे॒तत् ( ८ ) यूप॑स्य॒ यदूर्ध्वं च॒षाला॒त्तेषां॒ तद्भा॑-  
ग॒धेयं॒ ताने॒व तेन॑ प्रीणाति दे॒वा वै सꣳस्थि॑ते॒ सोमे॒  
प्र स्रुचोऽह॑र॒न्प्र यूपं॒ ते॑ऽमन्यन्त यज्ञवेश॒सं वा इ॒दं  
कु॑र्म॒ इति॒ ते प्र॑स्त॒रꣳ स्रु॒चां नि॒ष्क्रयणमश्य॒न्त्स्वरुं॒  
यूप॑स्य॒ सꣳस्थि॑ते॒ सोमे॒ प्र प्र॑स्त॒रꣳ हर॑ति जु॒होति॒  
स्वरु॒मय॑ज्ञवेशसाय ( ९ )।।

( द॒धा॒ति॒ प्रत्यृ॒चा सम॑र्धयेय॒मित्यू॒र्जैनं॒
