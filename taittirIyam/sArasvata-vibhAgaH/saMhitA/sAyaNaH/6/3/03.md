(अथ षष्ठाष्टके तृतीयप्रपाठके तृतीयऽनुवाकः )।  
वै॒ष्ण॒व्यर्चा हुत्वा यूप॒मच्छै॑ति वैष्ण॒वो वै  
द॒वतया॒ यूपऋ॒ दे॒वत॒याऽच्छै॒त्यत्य॒न्यानगां॒  
नान्यानुपा॑गा॒मित्या॒हाति॒ ह्य॑न्यानेति॒ नान्यानु॒पै-  
त्य॒र्वाक्त्वा॒ परै॑रविदं प॒रो॑ऽव॑रैरित्या॑हा॒र्वाग्घ्ये॑नं॒ परै॑-  
र्वि॒न्दति॑ प॒रो॑व॑रै॒स्तं त्वा॑ जुषे (१) वै॒ष्ण॒वं दे॑वय॒-  
ज्याया॒ इत्या॑ह देवय॒ज्यायै॒ ह्ये॑नं जु॒षते॑ दे॒वस्त्वा॑  
सवि॒ता मध्वाऽन॒क्त्वित्या॑ह॒ तेज॑सैवैन॑मनक्त्योष॑धे  
त्राय॑स्वैन॒ꣳ स्वधि॑ते॒ मैनꣳ॑ हिꣳसी॒रित्या॑ह वज्‍रो॒ वै  
स्वाधि॑तिः॒ शान्त्यै॒ स्वधि॑तेर्वृ॒क्षस्य॒ बिभ्य॑तः प्रथ॒मेन  
शक॑लेन स॒ह तेजः॒ परा॑ पतति॒ यः प्र॑थ॒मः शक॑लः  
परा॒प॒ते॒त्तमप्या ह॑रे॒त्सते॑जसम् ( २ ) ए॒वैन॒मा ह॑रती॒मे  
वै लो॒का यूपै॑त्प्रय॒तो बि॑भ्यति॒ दिव॒मग्‍रे॑ण॒ मा  
ले॑खीर॒न्तरि॑क्षं॒ मध्ये॑न॒ मा हिꣳ॑सी॒रित्या॑है॒भ्य  
ए॒वैनं॑ लो॒केभ्यः॑ शमयति॒ वन॑स्पते श॒तव॑ल्शो॒ वि  
रो॒हेत्या॒व्रश्च॑त्ते जुहोति॒ तस्मा॑दा॒व्रश्च॑नाद्वृ॒क्षाणां भू-  
या॑ꣳस॒ उत्ति॑ष्ठन्ति स॒हस्र॑वल्शा॒ वि व॒यꣳ रुपे॒मे-  
त्या॑हा॒ऽऽशिष॑मेवैतामा शा॒स्तेऽन॑क्षसङ्गम् (३) वृ॒श्चे॒-  
द्यद॑क्षस॒ङ्गं वृ॒श्चेद॑धई॒षं यज॑मानस्य प्र॒मायु॑कꣳ  
स्या॒द्य का॒मये॒ता॑प्रतिष्ठितः स्या॒दित्या॑रो॒हं तस्मै॑

प्रपा॰ ३ अनु॰ ३] कृष्णयजुर्वेदीयतैत्तिरीयसंहिता।  
( यूपखण्डनाभिधानम् )  
वृश्चेदे॒ष वै वन॒स्पती॑ना॒मप्र॑तिष्ठि॒तोऽप्र॑तिष्ठित ए॒व  
भ॑वति॒ यं का॒मये॑ताप॒शुः स्या॒दित्य॑प॒र्णं तस्मै॒  
शुष्का॑ग्रं वृश्चेदे॒ष वै वन॒स्पती॑नामश॒ब्यो॑ऽप॒शुरे॒व  
भृ॑वति॒ यं का॒मये॑त पशु॒मान्त्स्या॒दिति॑ बहुप॒र्णं तस्मै॑  
बहुशा॒खं वु॑श्चेदे॒ष वै ( ४ ) वन॒स्पती॑नां प॒शा॒व्यः॑  
पशु॒माने॒व भ॑वति॒ प्रति॑ष्ठितं वृश्चेत्प्रति॒ष्ठाका॑मस्यै॒ष वै  
वन॒स्पती॑नां॒ प्रति॑ष्ठितो॒ यः स॒मे भूम्यै॒ स्वाद्योने॑ रू॒ढः  
प्रत्ये॒व ति॑ष्ठति॒ यः प्र॒त्यङ्ङुप॑नत॒स्तं वृ॑श्चे॒त्स हि मेध॑-  
म॒भ्युप॑नतः॒ पञ्चा॑रत्निं तस्मै॑ वृश्चे॒द्यं का॒मये॒तोप॑न॒मु-  
त्त॑रो य॒ज्ञो न॑मे॒दिति॒ पञ्चा॑क्षरा प॒ङ्क्तिः पाङ्क्तो॑  
य॒ज्ञ उपै॑न॒मुत्त॑रो य॒ज्ञः ( ५ ) न॒म॒ति॒ षड॑रत्निं प्रति॒-  
ष्ठाका॑मस्य॒ षड्वा ऋ॒तव॑ ऋ॒तुष्वे॒व प्रति॑ तिष्ठति  
स॒प्तार॑त्निं प॒शुका॑मस्य स॒प्तप॑दा॒ शक्व॑री प॒शवः॒  
शक्व॑री प॒शूने॒वाव॑ रुन्धे॒ नवा॑रत्निं॒ तेज॑स्कामस्य  
त्रि॒वृता॒ स्तोमे॑न॒ संमि॑तं॒ तेजं॑स्त्रि॒वृत्ते॑ज॒स्व्ये॑व भ॑व॒त्ये-  
का॑दशारत्निमिन्द्रि॒यका॑म॒स्यैका॑दशाक्षरा त्रि॒ष्टुगि॑-  
न्द्रि॒यं त्रि॒ष्टुगि॑न्द्रि॒याव्ये॑व भ॑वति॒ पञ्च॑दशारत्निं॒  
भ्‍रातृ॑व्यवतः पञ्चद॒शो वज्‍रो॒ भ्‍रातृव्याभिभूत्यै स॒प्त-  
द॑शारत्निं प्र॒जाका॑मस्य सप्तद॒शः प्र॒जाप॑तिः प्र॒जा-  
प॑ते॒राप्त्या॒ एक॑विꣳशत्यरत्निं प्रति॒ष्ठाका॑मस्यैक-  
विꣳशः स्तोमा॑नां प्रति॒ष्ठा प्रति॑ष्ठित्या अष्टाश्रिर्भ-  
यत्यृ॒ष्टाक्ष॑रा गाय॒त्री तेजो॑ गाय॒त्री गा॑य॒त्री य॑ज्ञ-  
मु॒खं तेज॑सै॒व गा॑यत्रि॒या य॑ज्ञमु॒खेन॒ संनि॑तः ( ६ ) ।।
